üì° API Endpoints

üñºÔ∏è Image Processing

POST /api/remove-background

Remove background from an image

curl -X POST "http://your-server:8000/api/remove-background" \
  -F "file=@photo.jpg"

POST /api/remove-background-with-mockup

Remove background + generate pillow mockup in one request

curl -X POST "http://your-server:8000/api/remove-background-with-mockup?color=beige" \
  -F "file=@photo.jpg"

POST /api/mockup

Generate mockup from uploaded image (optionally remove bg first)

curl -X POST "http://your-server:8000/api/mockup?color=navy&remove_bg=true" \
  -F "file=@design.png" --output mockup.png

üì• Retrieve Images

GET /api/images

List all processed images

curl "http://your-server:8000/api/images"

GET /api/images/{id}/original

Get original uploaded image

curl "http://your-server:8000/api/images/1/original" --output original.jpg

GET /api/images/{id}/processed

Get background-removed image (PNG with transparency)

curl "http://your-server:8000/api/images/1/processed" --output processed.png

GET /api/images/{id}/mockup

Get pillow mockup. Colors: white, cream, beige, gray, black, navy, blush, sage

curl "http://your-server:8000/api/images/1/mockup?color=blush" --output mockup.png

DELETE /api/images/{id}

Delete an image and all associated files

curl -X DELETE "http://your-server:8000/api/images/1"

üìÑ PDF Export

GET /api/pdf/originals

Download PDF with all original images (one per page)

curl "http://your-server:8000/api/pdf/originals" --output originals.pdf

GET /api/pdf/processed

Download PDF with processed images. Layouts: simple, masonry, bento

curl "http://your-server:8000/api/pdf/processed?layout=masonry" --output gallery.pdf

GET /api/pdf/all

Download combined PDF with all images in masonry layout

curl "http://your-server:8000/api/pdf/all" --output complete.pdf
